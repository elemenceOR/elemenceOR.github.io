Projects

These are the most important projects I've worked on so far.
Also, I've learned and enjoyed a lot by doing these projects.

Prosthetic Arm

Develop an open-source embedded system to control the elbow joint of a prosthetic arm. FFNN was used to train the turning angle of the elbow and 3D scanning technology and Solidworks were used to design the prosthetic socket and the arm. 3D printing was used as the manufacturing process.

Pneumatic Engine

Designing an alternative mechanism to replace the traditional diesel generator using compressed air to generate electricity. SolidWorks was used for designing. The calculations were done considering real situations

Smart Green House

Monitor and control changes in conditions such as temperature, humidity, the soil moisture level in a scaled-down model of a greenhouse using sensors, actuators, and a Bluetooth module used to transfer the data to a mobile application which also implements an intruder detection system.
